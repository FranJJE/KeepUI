# KeepUI

> Angular cross-platform UI component library with optional Capacitor support.

This repository contains the **KeepUI** Angular workspace — a reusable component library
(`@keepui/ui`) that works in both standard Angular web applications and Angular + Capacitor
mobile/hybrid applications, without coupling UI components to native APIs.

## Quick Start

```bash
# Install dependencies
npm install

# Build the library (production, APF)
npm run build

# Run unit tests
npm test
```

## Packages

| Package | Description |
|---|---|
| `@keepui/ui` | Core components, ports, tokens, web adapter |
| `@keepui/ui/capacitor` | Optional Capacitor adapter for native image picking |
